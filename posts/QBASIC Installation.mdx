Download both files from this link: https://linkgenie.me/s/CTEAP  
Then install the setup file of DOSBOX  
And extract the rar file by "Extract Here" button  
Then copy the extracted folder to C:\oldDOS\  
After creating the oldDDOS in C:\  
Then search for DOSBOX in Windows search and open the app  
And type "mount c c:\oldDOS\QB45\"  
Then press enter and type "c:",press enter  
Then type QB to open Qbasic
