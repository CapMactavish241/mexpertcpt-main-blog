Download both files from this link: https://linkgenie.me/s/CTEAP <br>
Then install the setup file of DOSBOX <br>
And extract the rar file by "Extract Here" button <br>
Then copy the extracted folder to C:\oldDOS\ <br>
After creating the oldDDOS in C:\ <br>
Then search for DOSBOX in Windows search and open the app <br>
And type "mount c c:\oldDOS\QB45\" <br>
Then press enter and type "c:",press enter <br>
Then type QB to open Qbasic <br>
